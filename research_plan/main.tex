\section{Research Purpose}
The purpose of this research is to understand the compatibility between message-passing frameworks, such as k-machine model, congested clique and BSP, and distributed computing framework, such as MPI, OpenMP, Pytorch and tensorflow.

How to evaluate compatibility /incompatibility
\begin{enumerate}
  \item{
    Evaluating Performance.
      \begin{itemize}
        \item How does this framework perform compared to another framework with the same message-passing model and problem (Communication rounds, memory usage).
        \item How does this framework perform with a model if the problem scales (Strong scaling and weak scaling).
      \end{itemize}
    }
  \item {
    System requirement from the programming framework.
      \begin{itemize}
        \item memory usage
        \item CPU usage
        \item network bandwidth
        \item disk I/O
      \end{itemize}
    }
\end{enumerate}

\section{Objective}
For a framework and a model, the objective would be to show either of the following:
\begin{itemize}
  \item The framework and model is perfectly compatible.
  \item The framework and model is not perfectly compatible but with some extra work, the model and framework could work together to solve problems in a relative efficient way.
  \item The framework and model is not compatible at all.
\end{itemize}
To show the compatibility, algorithms will be presented so as proofs. To show the incompatibility, proofs need to be given.
\section{Method}
\begin{itemize}

  \item Reading documentation of programming frameworks
  \item {
    Analyzing the compatibility of frameworks and models:
      \begin{enumerate}
        \item Choosing classical problems that requires HPC (Graph problems).
        \item Designing algorithms using frameworks and models.
        \item Analyzing performance from time and memory-wise and consider scaling.
        \item Conclude the compatibility between a model and a framework.
        \item (Optional) Applying the algorithm on real machines to verify the result.
      \end{enumerate}
  }
\end{itemize}
